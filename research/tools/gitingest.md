## What it does

Replace "hub" with "ingest" in any GitHub URL to get a prompt-friendly extract. Also available as a Python package and CLI. Provides file/directory stats, token counts, and smart formatting. Browser extensions for Chrome and Firefox.

## Strengths

- Zero-install web interface
- URL-replacement UX is brilliant for quick use
- Python package for programmatic use

## Limitations

- Web version only works on public repos
- Less configurable than Repomix
- No compression/Tree-sitter features
- One-shot

## Relevance to chat-spec

Quick-and-dirty context tool. Not in the same problem space as chat-spec.
