# code2prompt

- **Repo:** [mufeedvh/code2prompt](https://github.com/mufeedvh/code2prompt)
- **Website:** [code2prompt.dev](https://code2prompt.dev/)
- **Stars:** ~5,100
- **Language:** Rust

## What it does

Rust CLI tool to convert codebases into LLM prompts with source tree visualization, Handlebars templating, token counting, and glob-based filtering. Also available as an MCP server.

## Strengths

- Fast (Rust)
- Handlebars templating gives full control over output format
- TUI for interactive configuration
- MCP server mode

## Limitations

- Smaller community than Repomix
- Multiple forks/reimplementations create naming confusion (Python version by raphaelmansuy, Node.js versions)
- One-shot

## Relevance to chat-spec

Context packing tool, not in the same space.
